## 📖 Descripción

Este proyecto consiste en un página web la cual nos permite mostrar los animalitos que se encuentran en estos smomentos buscando un hogar una familia.

## ✨ Características

- Buscar mascotas.

## 📦 Instalación

1. Clonar el repositorio

```bash
git clone https://github.com/MishelleBohorquez/TIFC2DEV-SDPF6-Simulaci-n-de-Proyecto-SCRUM.git
```

2. Entrar al proyecto

```bash
cd TIFC2DEV-SDPF6-Simulaci-n-de-Proyecto-SCRUM

```

3. Ejecutar

```bash
node inicio.html
```
